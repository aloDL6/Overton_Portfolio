Game On Project Process

Our project focus was to use HTML, CSS, and Javascript to create a rudimentary game. Using the code template of PIG, provided by the professor Bill Mead, then adding complexity on our own.

The game of PIG is simple, you have two oppenents who roll a pair of die to obtain points until someone reachesor surpasses a set number goal. After each roll, the player can choose to either roll again or pass the die to the other player. However, if the player rolls a 1 on either die, they forfeit their turn to the other player and if they roll snake eyes, or two 1s, they lose all their points and must start back at 0.

Conceptually, I wanted my version of the game to be a sort of battle RPG with a story behind in, not a traditional set up for a board game. The players aim to defeat monsters using the point totals of their die, whoever defeats their monster first wins, while the other loses to the monster. The basic mechanics of the game itself is the same, but I added a level of customization such as player names, being able to set your own goal, and sound effects and music.
