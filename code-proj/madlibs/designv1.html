MadLibs Project Process

Final Project

Our first project was focused on user input and retaining that data throughout the JavaScript. To do so, we would create a madlib, allowing users to type in their own words, then to use code to use this information within the story we write.

In my concept, I wanted to create a sort of news story. Inspired by sequences in the Tomodachi Life games, the news are chaotic and absurd, making it an intriguing news story whatever words the user inputs.
